Sistema de Login y Sign Up desarrollado para PCJ Representaciones Médicas, empresa dedicada a la comercialización de insumos médicos. La solución implementa autenticación de usuarios mediante Firebase, integrando principios de programación orientada a objetos (OOP) para la organización de la lógica de negocio. La interfaz fue construida con HTML y CSS, priorizando claridad visual, usabilidad y estructura modular.
